# Быстрый тест QR-кодов

## Проблема: Backend не может подключиться к базе данных

Для работы мобильного приложения нужен запущенный backend с базой данных.

## Решение 1: Запуск через Docker (рекомендуется)

```bash
# 1. Запустите Docker Desktop на Mac

# 2. Запустите все сервисы
cd "/Users/kuralbekadilet475/Tutas Ai"
make up

# 3. Подождите 30 секунд для инициализации
sleep 30

# 4. Создайте тестовую трубу
python3 scripts/create_test_pipe.py --company COMPANY

# 5. Сгенерируйте QR-код
python3 scripts/generate_qr.py COMPANY
```

## Решение 2: Использование существующих QR-кодов

Если у вас уже есть QR-коды в папке `qr_codes/`, используйте их:

1. Откройте папку `qr_codes/`
2. Найдите файл PNG с QR-кодом
3. Отсканируйте его в мобильном приложении

**Важно:** Труба с таким QR-кодом должна существовать в базе данных!

## Решение 3: Веб-генератор QR-кодов

Откройте в браузере: `http://localhost:3000/qr-generator.html`

Там можно:
- Сгенерировать QR-код
- Скачать его
- Использовать для тестирования

## Проверка работы

1. **Проверьте backend:**
   ```bash
   curl http://localhost:8000/health
   ```
   Должен вернуть: `{"status":"healthy","service":"backend-api"}`

2. **Проверьте список труб:**
   ```bash
   curl -H "Authorization: Bearer dev-api-key-12345" \
        http://localhost:8000/api/v1/pipes
   ```

3. **Создайте трубу:**
   ```bash
   python3 scripts/create_test_pipe.py
   ```
